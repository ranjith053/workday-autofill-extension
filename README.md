# 🚀 Workday Autofill Chrome Extension

A lightweight Chrome Extension that automates repetitive Workday job applications by autofilling work experience and education.

---

## ✨ Features

- Auto-fill Work Experience
- Auto-fill Education
- Save multiple jobs
- Local Chrome Storage
- Fast and lightweight
- Built with Manifest V3

---

## 🛠 Tech Stack

- JavaScript
- HTML
- CSS
- Chrome Extensions API

---

## 📦 Installation

1. Clone the repository

```bash
git clone https://github.com/ranjith053/workday-autofill-extension.git
```

2. Open Chrome

3. Visit

```
chrome://extensions
```

4. Enable **Developer Mode**

5. Click **Load unpacked**

6. Select this project folder
